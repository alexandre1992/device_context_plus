# device_context_plus_example

Demonstrates how to use the device_context_plus plugin.

## Getting Started

A Flutter plugin that provides system and runtime context for apps.

## Features

- ✅ iOS Team ID
- ✅ Android installer store
- ✅ App version and build
- ✅ Device info (model, OS)
- ✅ Locale and timezone

## Usage

```dart
final info = await DeviceContext.getAll();

```
